pica.js — Upscaling de înaltă calitate

✓ activ în pagina ta

Mărește imaginile (2x, 3x, 4x) fără a le face pixelate. Folosește algoritmul Lanczos3, același ca Photoshop. Deja integrat și funcțional în secțiunea Upscaler 4x.

Este deja în fișierul tău — linia din <head>:
<script src="https://cdn.jsdelivr.net/npm/pica@9.0.1/dist/pica.min.js"></script>
Dacă vrei să o folosești și în altă pagină:
1
Deschide fișierul HTML nou cu Notepad / VS Code
2
Găsește linia </head> și adaugă ÎNAINTE de ea:
<script src="https://cdn.jsdelivr.net/npm/pica@9.0.1/dist/pica.min.js"></script>
3
Gata. Acum în codul tău JS poți scrie:
const p = pica(); await p.resize(canvasSursa, canvasDestinatie, { quality: 3 });
💡 quality: 3 = Lanczos3, cel mai bun. quality: 1 = rapid dar mai slab.
→ Documentație completă: github.com/nodeca/pica

ExifReader.js — Citire metadate RAW/JPEG

✓ activ în pagina ta

Citește toate informațiile dintr-o fotografie: cameră, obiectiv, ISO, timp expunere, GPS și altele. Funcționează cu RAW (CR2, NEF, ARW, DNG) și JPEG. Deja integrat în secțiunea RAW & EXIF.

Este deja în fișierul tău — linia din <head>:
<script src="https://cdn.jsdelivr.net/npm/exifreader@4.21.1/dist/exif-reader.min.js"></script>
Dacă vrei să o folosești și în altă pagină:
1
Adaugă în <head>:
<script src="https://cdn.jsdelivr.net/npm/exifreader@4.21.1/dist/exif-reader.min.js"></script>
2
Folosești cu un input[type=file]:
const reader = new FileReader(); reader.onload = e => { const tags = ExifReader.load(e.target.result, { expanded: true }); console.log(tags.exif.Model.description); // ex: "Canon EOS R5" console.log(tags.exif.ISOSpeedRatings.value); // ex: 400 }; reader.readAsArrayBuffer(fisier);
→ github.com/mattiasw/ExifReader

CamanJS — Filtere foto (vintage, lomo etc.)

✓ activ în pagina ta

Aplică filtere foto pe un canvas: luminozitate, contrast, saturație, și preseturi gata-făcute ca vintage, lomo, sunrise, clarity și altele. Deja inclus în pagina ta, îl poți folosi oricând.

Este deja în fișierul tău:
<script src="https://cdnjs.cloudflare.com/ajax/libs/camanjs/4.1.2/caman.full.min.js"></script>
Cum îl folosești pe un canvas cu imagine:
1
Ai un canvas cu o imagine desenată pe el (id = "myCanvas")
2
Apelezi Caman cu ajustările dorite:
Caman('#myCanvas', function() { this.brightness(10) // -100 la +100 .contrast(5) .saturation(15) .exposure(5) .render(); });
3
Sau aplici un preset gata-făcut:
Caman('#myCanvas', function() { this.sunrise().render(); });
💡 Preseturi disponibile: sunrise · vintage · lomo · clarity · sinCity · crossProcess · pinhole · oldBoots · glowingSun · hazyDays · herMajesty · nostalgia · hemingway · concentrate
⚠ Atenție: CamanJS modifică canvas-ul async. Dacă vrei să descarci rezultatul, fă-o în callback-ul după render():

this.render(function() { /* descarcă canvas-ul aici */ });
→ github.com/meltingice/CamanJS

Cropper.js — Crop, Rotire, Flip

✓ activ în pagina ta

Crop interactiv cu mouse-ul, rotire cu orice unghi, flip orizontal/vertical, și rapoarte fixe (1:1, 16:9 etc.). Rezultatul este la rezoluția originală. Deja integrat în secțiunea Crop / Rotate.

Cum funcționează Cropper.js — pași simpli:
1
Ai nevoie de DOUĂ lucruri: fișierul CSS și fișierul JS, ambele deja în pagina ta:
<!-- în head --> <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/cropperjs@1.6.2/dist/cropper.min.css"> <!-- înainte de /body --> <script src="https://cdn.jsdelivr.net/npm/cropperjs@1.6.2/dist/cropper.min.js"></script>
2
Pui o imagine în HTML:
<img id="poza" src="foto.jpg" style="max-width:100%">
3
Activezi cropper-ul în JS:
const cropper = new Cropper(document.getElementById('poza'), { aspectRatio: 16 / 9, // sau NaN pentru liber viewMode: 1 });
4
Descarci zona selectată la rezoluție originală:
const canvas = cropper.getCroppedCanvas(); canvas.toBlob(blob => { const a = document.createElement('a'); a.href = URL.createObjectURL(blob); a.download = 'crop.jpg'; a.click(); }, 'image/jpeg', 1.0);
💡 getCroppedCanvas() fără argumente returnează imaginea la rezoluția ORIGINALĂ, nu la cea afișată pe ecran.
→ github.com/fengyuanchen/cropperjs

Compressor.js — Micșorare fișier imagine

✓ activ în pagina ta

Reduce dimensiunea unui fișier JPG/PNG/WebP înainte de a-l urca pe un site. De exemplu, un fișier de 8MB devine 800KB cu calitate vizuală identică. Totul se face în browser, fără server. Deja integrat în secțiunea Compresie.

Adaugă în <head> dacă vrei în altă pagină:
<script src="https://cdn.jsdelivr.net/npm/compressorjs@1.2.1/dist/compressor.min.js"></script>
Cum îl folosești — extrem de simplu:
1
Utilizatorul selectează un fișier din input[type=file]
2
Apelezi Compressor cu fișierul:
new Compressor(fisier, { quality: 0.8, // 0.1 (mic) → 1.0 (original) maxWidth: 3000, // redimensionează dacă e mai lat mimeType: 'image/jpeg', success(rezultat) { // rezultat = fișier comprimat console.log('Înainte:', fisier.size, 'bytes'); console.log('După:', rezultat.size, 'bytes'); // descarcă: const a = document.createElement('a'); a.href = URL.createObjectURL(rezultat); a.download = 'foto-mic.jpg'; a.click(); }, error(e) { alert(e.message); } });
💡 Calitate recomandată pentru imobiliare: 0.82 (82%) — reduce fișierul cu ~75% fără pierdere vizibilă.
→ github.com/fengyuanchen/compressorjs

Cum pui fișierul pe GitHub Pages

ghid pas cu pas

GitHub Pages este un serviciu gratuit care transformă un fișier HTML în site accesibil de oriunde cu un link. Nu ai nevoie de hosting, nu plătești nimic.

Prima dată — creare repository:
1
Mergi la github.com și fă cont dacă nu ai
2
Apasă butonul verde New (sau + din colțul dreapta sus → New repository)
3
Denumește repository-ul, ex: re-photo-editor — bifează Public → apasă Create repository
Upload fișier index.html:
4
În repository-ul nou, apasă Add file → Upload files
5
Trage fișierul index.html în zona de upload
6
Apasă Commit changes (butonul verde jos)
Activare GitHub Pages:
7
Mergi la Settings (tab-ul din meniul repository-ului)
8
În meniul din stânga apasă Pages
9
La Source selectează Deploy from a branch, branch = main, folder = / (root) → apasă Save
10
Aștepți ~1 minut, refresh la pagina Settings/Pages și vei vedea link-ul tău:
https://NUMELE-TAU.github.io/re-photo-editor/
💡 Actualizare după modificări: Mergi la repository → Add file → Upload files → urci noul index.html care înlocuiește vechiul → Commit. Site-ul se actualizează automat în ~30 secunde.
✅ Librăriile CDN funcționează pe GitHub Pages fără nicio problemă — se încarcă de pe internet direct în browser-ul vizitatorului.
