Librării JS — Instalare

Toate librăriile sunt deja active

Fișierul tău index.html conține linkuri CDN care încarcă automat librăriile de pe internet. Nu trebuie să descarci sau instalezi nimic — urci doar index.html pe GitHub.

pica.js — Upscaler Lanczos3 · Folosit la Upscaler 4x
ExifReader.js — Citire EXIF/RAW · Folosit la RAW & EXIF
Cropper.js — Crop/Rotate/Flip · Folosit la Crop
Compressor.js — Compresie · Folosit la Compresie
💡 Librăriile CDN funcționează pe GitHub Pages fără nicio problemă suplimentară. Ai nevoie de conexiune internet când folosești pagina.
⤡

pica.js — Upscaling Lanczos3

✓ activ

Mărește imagini 2x-4x fără pixelare. MIT License.

Deja în fișierul tău:
<script src="https://cdn.jsdelivr.net/npm/pica@9.0.1/dist/pica.min.js"></script>
Cum se folosește:
const p = pica(); await p.resize(srcCanvas, dstCanvas, { quality: 3 });
→ github.com/nodeca/pica
🏷

ExifReader.js — EXIF din RAW/JPEG

✓ activ

Citește cameră, obiectiv, ISO, GPS din orice format. MIT License.

Deja în fișierul tău:
<script src="https://cdn.jsdelivr.net/npm/exifreader@4.21.1/dist/exif-reader.min.js"></script>
Cum se folosește:
const reader = new FileReader(); reader.onload = e => { const tags = ExifReader.load(e.target.result, { expanded: true }); console.log(tags.exif.Model.description); }; reader.readAsArrayBuffer(fisier);
→ github.com/mattiasw/ExifReader
✂

Cropper.js — Crop avansat

✓ activ

Crop interactiv, rotire, flip, aspect ratio fix. MIT License.

Deja în fișierul tău (CSS + JS):
<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/cropperjs@1.6.2/dist/cropper.min.css"> <script src="https://cdn.jsdelivr.net/npm/cropperjs@1.6.2/dist/cropper.min.js"></script>
Cum se folosește:
const cropper = new Cropper(imgElement, { aspectRatio: 16/9 }); const canvas = cropper.getCroppedCanvas(); // rezoluție originală
→ github.com/fengyuanchen/cropperjs
📦

Compressor.js — Compresie

✓ activ

Reduce fișierul cu 60-90% fără pierdere vizibilă. MIT License.

Deja în fișierul tău:
<script src="https://cdn.jsdelivr.net/npm/compressorjs@1.2.1/dist/compressor.min.js"></script>
Cum se folosește:
new Compressor(file, { quality: 0.8, success(result) { // result = Blob comprimat } });
→ github.com/fengyuanchen/compressorjs
🚀

Cum pui pe GitHub Pages

ghid pas cu pas
Prima dată:
1
Mergi la github.com → fă cont dacă nu ai
2
Apasă New repository → denumire ex: re-photo-editor → bifează Public → Create
3
Apasă Add file → Upload files → trage fișierul index.html → Commit changes
4
Mergi la Settings → Pages → Source: main / root → Save
5
Aștepți 1 minut → link-ul apare:
https://NUMELE-TAU.github.io/re-photo-editor/
💡 Actualizare: upload nou index.html → Commit → site se actualizează în ~30 sec.
