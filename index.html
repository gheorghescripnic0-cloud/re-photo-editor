Open Source Tools — Integrare & Resurse

🌅

Darktable — Editor RAW complet (gratuit, open-source)

Alternativa gratuită la Lightroom. Suportă toate formatele RAW, HDR merge, tethering, batch export. Disponibil pe Windows, Mac, Linux.

→ darktable.org
📦

RawTherapee — RAW processing avansat

Procesare RAW de nivel profesional cu HDR, retinex, wavelet denoising. 100% gratuit.

→ rawtherapee.com
🖼

GIMP + G'MIC — Editare avansată (gratuit)

GIMP cu pluginul G'MIC oferă HDR tone mapping, sky replacement, inpainting, deblur prin blind deconvolution, upscaling AI (ESRGAN). Instalează G'MIC din Filters > G'MIC.

→ gmic.eu
🤖

Real-ESRGAN — Upscaling AI (4x, 8x)

Cel mai bun upscaler AI open-source. Rulează local sau online. Produce rezultate spectaculoase pe poze imobiliare — detalii clare, fără artefacte.

→ github.com/xinntao/Real-ESRGAN
☁

SkyFilter — Sky Replacement AI (open-source Python)

Script Python care înlocuiește cerul automat cu AI. Mult mai precis decât metodele bazate pe culoare. Rulează local fără cont.

→ github.com/xiaohu2015/SkyFilter
🔍

Upscayl — AI Upscaler desktop (gratuit)

Aplicație desktop care folosește Real-ESRGAN. Drag & drop, batch processing, 4x upscale cu un click. Ideal pentru batch de poze imobiliare.

→ upscayl.org
🎨

Luminance HDR — HDR merge dedicat

Software dedicat pentru HDR merge cu algoritmi multipli: Mantiuk, Fattal, Drago, Reinhard. Ideal pentru real estate interior cu ferestre.

→ qtpfsgui.sourceforge.net
🧹

Cleanup.pictures — Object Removal (gratuit online)

Elimină obiecte nedorite din poze (mașini, cabluri, obiecte). Funcționează direct în browser, 100% gratuit pentru poze mici.

→ cleanup.pictures
✂

Remove.bg / Rembg — Background removal

Rembg este versiunea open-source ce rulează local (Python). Elimină fundalul pentru fotografii de produse sau portrete. Gratuit fără limite local.

→ github.com/danielgatis/rembg
📐

Perspective Correction — deskew.js

Corectare perspectivă pentru poze imobiliare (linii verticale convergente). Librărie JS open-source, integrabilă în browser.

→ github.com/evanw/skew
⚡

Squoosh — Compresie imagine (Google, open-source)

Compresie avansată în browser: WebP, AVIF, MozJPEG, OxiPNG. Reduce dimensiunea fișierelor fără pierdere vizibilă de calitate. Perfect pentru listing-uri imobiliare.

→ squoosh.app
